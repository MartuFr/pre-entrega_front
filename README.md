🌸E-Commerce de Maquillaje - Proyecto Front-End🌸
Este proyecto es un e-commerce dedicado a la venta de productos de maquillaje. Fue desarrollado como parte de un curso de Desarrollo Web Front-End con el objetivo de desarrollar buenas prácticas de código.

🎯 Objetivo del Proyecto
El foco principal de este desarrollo fue construir una interfaz atractiva, funcional y accesible utilizando HTML y CSS, aplicando los conocimientos adquiridos durante la cursada.

✨ Características Principales
Este proyecto destaca por una estructuración rigurosa basada en etiquetas semánticas de HTML5, lo que garantiza una web accesible. A nivel visual, presenta un diseño moderno que utiliza CSS Grid y Flexbox para organizar los productos en tarjetas interactivas, complementadas con transiciones suaves, tipografías de Google Fonts y elementos multimedia envolventes, como un banner con video de fondo en reproducción continua. Finalmente, la experiencia del usuario se completa con una página de contacto que incluye un formulario 100% funcional impulsado por Formspree, lo que permite recibir los mensajes de los clientes directamente sin necesidad de programar un backend.
