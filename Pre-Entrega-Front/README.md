Proyecto e commerce

Descripción:
Logré hacer esta página web de e-commerce orientada al rubro del maquillaje con el objetivo de reflejar y poner en práctica todas las herramientas y buenas prácticas abordadas durante la cursada.

Para la construcción del sitio, aseguré una arquitectura limpia y accesible utilizando HTML semántico, aplicando correctamente etiquetas fundamentales como <header>, <main> y <footer>. El diseño visual fue resuelto con CSS, donde uno de los desafíos que pude resolver gracias a las clases fue el dominio de Flexbox y CSS Grid. Entender estas herramientas me permitió estructurar el layout y organizar las tarjetas de los productos de manera precisa.Tambien me apoyé en el uso de Copilot como asistente de programación, lo cual fue clave para agilizar tareas mecánicas como la repetición de código en las cards y para resolver la lógica de animaciones CSS , como la cinta de promociones en movimiento. Asimismo, complementé la interfaz gráfica integrando tipografías de Google Fonts y un video de fondo continuo que aportan dinamismo al proyecto.

A través del uso de Formspree, logré habilitar el envío real de correos electrónicos. Para ello, generé un proyecto en su sitio web, especifiqué mi dirección de correo de recepción y vinculé mi formulario HTML. Esto me permitió capturar los mensajes de los usuarios que completaran el form.
